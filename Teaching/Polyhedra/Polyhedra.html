Polyhedra

This reading group was done with an undergraduate during Winter 2024 as part of the WDRP. The description of the project can be found here. This project was designed as a friendly survey of many upper-level topics, including topology, graph theory, abstract algebra and combinatorics; using polyhedra as the object of study.

The main reference we used was:

"Polyhedra" by Peter R. Cromwell
